Проект реализован для вычисления в математике промежуточных значений величин по имеющемуся дискретному набору известных значений.

Функционалтность проекта:

1.Добавление нового узла нажатием левой кнопки мыши.

2.Выделение выбранного узла и его параметры.

3. Изменение параметров с помощью соответсвующих спинбоксов:
X - координата по оси X.
Y - координата по оси Y.
T - натяжение. Изменяет длину касательного вектора.	
B - предвзятость. В первую очередь изменяет направление касательного вектора.
C - непрерывность. Изменяет резкость при изменении касательных.

4. Удаление определенного узла реализуется правой кнопкой мыши.

5. Добавление нового узла между определенными двумя.

6. Выбор сплайна:
	а - Kochanek-Bartels - сплайн Кочана — Бартельса 
	б - Bezier - Кривая Безье, при этом изменение параметров (t,b,c) не осуществляется
	в - Poline - ломанная, при этом изменение параметров (t,b,c) не осуществляется.

6. Представлено меню:
	File->Save - сохранение файла с сплайном (Горячие клавиши Ctr+S).*
	File->Open - откоытие файла с сплайном (Горячие клавиши Ctr+O).*
	File->Close - закрытие проекта
	
	Edit->Undo - реализуется отмена последенего действия до 30 последних (Горячие клавиши Ctr+Z).*
	Edit->Redo - реализуется повторение отмененного действия до 30 последних (Горячие клавиши Shift+Ctr+Z).*

	About - информация об авторе

*Горячие клавиши срабатывают при нажатии на главное меню.
